# 附件上传问题

不能上传附件，百度了下，发现typecho有个检测服务器环境是不是SAE，BAE，ACE及是否是国内常见的虚拟主机，如果是则不让上传附件。尼玛坑啊，说多了都是泪，解决方案如下：找到文件typecho/var/Typecho/Common.php把下面的这段代码给它return false;然后再找到/typecho/usr/uploads文件夹，赋予777权限【chmod 777 uploads】

public static function isAppEngine()
{
	return false;
	/*
    return !empty($_SERVER['HTTP_APPNAME'])                     // SAE
    || !!getenv('HTTP_BAE_ENV_APPID')                       // BAE
    || !!getenv('SERVER_SOFTWARE')                          // BAE 3.0
    || (ini_get('acl.app_id') && class_exists('Alibaba'))   // ACE
    || (isset($_SERVER['SERVER_SOFTWARE']) && strpos($_SERVER['SERVER_SOFTWARE'],'Google App Engine') !== false) // GAE
     ;
	*/
}

# 不能加载其他网站图片问题

以前很多博客是在博客园或CSDN等平台写的，所以要转移过来，我使用的HTML转Markdown工具(插播一条广告，这个工具很好用)，但发现图片挂了，报403错误，查了下资料，要在文章页面(一般是post.php)头部加入<meta name="referrer" content="no-referrer"/>就可以了。同样可以用在https网站引用非https资源加载不出来的情况下。

# 给文本加样式

markdown格式不支持颜色，typecho可以直接输出html，写法如下，html使用!!!包括起来。

!!!
<p style="color:red">
	我是红色文字。
</p>
!!!
