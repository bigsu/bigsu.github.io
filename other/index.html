# win10

# 关闭chrome跟随windows启动：

win10重启后，一些在关机前没有关闭的应用会在下次开机自动打开

“设置-账户-登陆选项-隐私”把“更新或重启后，使用我的登录信息自动完成设备设置。”关掉

# 打开运行命令(win+R)历史记录：

进入设置->隐私->打开 允许windows跟踪应用启动，已改进开始和搜索结果

# 设置chrome保存mht单文件

打开chrome浏览器，地址栏输入：chrome://flags/ 后回车，Ctrl+F 搜索 MHTML,并找到 将网页另存为MHTML，点击 启用
